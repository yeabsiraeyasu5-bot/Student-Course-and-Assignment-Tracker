# 📚 Student Course & Assignment Tracker

A simple and user-friendly web application designed to help students organize their courses, manage assignments, and keep track of important deadlines. This project was developed as a school project using **HTML, CSS, and JavaScript**.

## ✨ Features

* 📖 Add, edit, and delete courses.
* 📝 Manage assignments and course details.
* 🔍 Search courses quickly.
* 🗂️ Filter courses by status (All, Pending, Completed).
* 📅 Track assignment deadlines and identify overdue tasks.
* 📊 View progress and completion statistics.
* 👤 Personal profile page with customizable user information.
* 🌙 Dark mode support.
* 💾 Automatic data saving using the browser's Local Storage.

## 🛠️ Technologies Used

* **HTML5** – Structure and content.
* **CSS3** – Styling and responsive layout.
* **JavaScript (ES6)** – Interactivity and application logic.
* **Local Storage API** – Persistent data storage without a database.

## 🚀 How to Use

1. Open the website in your browser.
2. Add your courses and assignment information.
3. Use the search and filter options to organize your tasks.
4. Track your progress and monitor upcoming deadlines.
5. Your data will be saved automatically in your browser using Local Storage.

## 🎯 Purpose of the Project

The goal of this project is to provide students with a simple tool for organizing their academic activities. It demonstrates the use of front-end web technologies to create an interactive and practical application without requiring a backend server or database.
